# Web Shortener App

## Overview
The Web Shortener App is a simple web application that allows users to shorten long URLs, manage their profiles, and access their accounts securely. The application consists of three main pages: a login page, a home page featuring the URL shortener tool, and a user profile page.

## Project Structure
```
web-shortener-app
├── index.html        # Main entry point for the URL shortener tool
├── login.html        # Login page for user authentication
├── user.html         # User profile page displaying user information and short URLs
├── css
│   └── styles.css    # Styles for the application
├── js
│   ├── app.js        # Main JavaScript logic for the home page
│   ├── auth.js       # User authentication functions
│   └── profile.js     # Functions to manage user profile data
└── README.md         # Project documentation
```

## Features
- **User Authentication**: Users can log in to their accounts to access personalized features.
- **URL Shortening**: Users can input long URLs and generate short URLs easily.
- **User Profiles**: Each user has a profile page displaying their short URLs, corresponding long URLs, and expiry information.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to access the application.
3. Use the login page to authenticate your account or create a new account if needed.
4. Navigate to the home page to use the URL shortener tool.
5. Access your user profile to view your shortened URLs and their details.

## Usage Guidelines
- Ensure you have a modern web browser for the best experience.
- Follow the prompts on each page to navigate through the application.
- For any issues or feature requests, please refer to the project's issue tracker.
